# 🌦️ Climate Data EDA & Weather Prediction

A data science mini project focused on exploratory data analysis (EDA) and weather prediction using Python and Jupyter Notebook.

---

## 📁 Repository Structure

```
datascience/
├── Climate.ipynb                      -   Main Jupyter Notebook (EDA + Prediction)
├── climate.csv                         -  Dataset used for analysis
└── Climate_Data_EDA_Project_Report.pdf  - Full project report
```

---

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a climate dataset to uncover patterns and trends in weather data, followed by predictive modeling to forecast weather conditions.

### Key Steps:
- Data loading and cleaning
- Statistical summary and missing value handling
- Visualizations (temperature trends, rainfall distribution, correlations)
- Feature engineering
- Predictive modeling for weather forecasting

---

## 🛠️ Technologies Used

- **Python 3**
- **Jupyter Notebook**
- **Pandas** – Data manipulation
- **NumPy** – Numerical operations
- **Matplotlib / Seaborn** – Data visualization
- **Scikit-learn** – Machine learning models

---

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/karthik11239a097/datascience.git
   cd datascience
   ```

2. **Install dependencies**
   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn jupyter
   ```

3. **Run the notebook**
   ```bash
   jupyter notebook Climate.ipynb
   ```

---

## 📊 Dataset

The `climate.csv` file contains historical weather/climate data with features such as temperature, humidity, rainfall, wind speed, and more.
